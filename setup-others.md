## Firefox Theme:
- Menu Help => More Troubleshooting Information
- Profile Folder row => click on Show in Finder
- Copy firefox contents (chrome, updater.sh, user.js, user-overrides.js, ...) to Firefox Profile Folder
- chmod +x updater.sh
- Run updater.sh
